Online website Builder

=============================================================

Tento online nástroj je určený pre používateľov, ktorých účelom je vytvoriť statickú web stránku. Používatelia tohto nástroja
nepotrebujú disponovať znalosťami z oblasti tvorby webových stránok. Tieto znalosti sú však výhodou pri práci s týmto nástrojom.
Samotná tvorba webu je veľmi rýchla. Pokiaľ má používateľ predpripravený všetok multimediálny obsah, ktorý chce prezentovať, je
možné pomocou tohto nástroja vytvoriť a publikovať webovú stránku v priebehu niekoľko desiatok minút. Chceme upozorniť, že pre
správne fungovanie tohto nástroja je potrebné mať povolený JavaScript v používanom prehliadači.
